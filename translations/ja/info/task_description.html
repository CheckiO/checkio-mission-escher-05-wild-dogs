爆風のほこりが収まるとあなたは領内に入っていきました。 しかし遠くから何やら鋭い遠吠えが聞こえてきました。 近寄ってみるとそれが大きくて黒い毛むくじゃらの少数の犬の群れであることがわかりました。 それらはむしろオオカミに近い獣のようであり、あなたの行く手をはばんでいました。 かつて招かれざる訪問者から城を守っていた番犬の子孫が野生化したものと思われます。 良い知らせはあなたはライフルを持っていることです。悪い知らせはそれほど多くの弾丸を持っていないことです。 あなたは生き残るために最良の射撃位置を探す必要があります。 各射撃の際に上手い位置取りをして下さい。

犬たちの位置が入力値として与えられます。 あなたの仕事は一度の射撃で最多の犬を殺せる直近の位置までの距離を計算することです。 (同一の直線上にいる犬は一度の射撃で全て殺せます) あなたの開始時の座標は (0, 0) です。

もし答えが 整数なら int型で返して下さい。 その他の場合は四捨五入して小数点第 2位までの float型で返して下さい。
一直線上にあなたを挟んで前後に犬が存在する様な場合 (例: 犬 犬 あなた 犬) を想定する必要はありません。 このようなテストケースは存在しません。

入力: 犬たちの座標

出力: 最良の射撃位置までの距離

例:

          [10, 173], [11, 190], [-100, 1]]) == 0.18

どのように使われるか: 部隊の配置や戦術の分析に利用できます。

事前条件:
2 <= amount of the dogs <= 15
0 <= distance to the right place <= 20
